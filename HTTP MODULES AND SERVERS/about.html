About Our Project

This is a simple website built with Node.js to demonstrate how to serve HTML files using the built-in http and fs modules.

The project was created as a learning exercise to understand the fundamentals of web servers without relying on frameworks like Express.js.

How It Works

This website uses:

  • Node.js http module to create a server
  • File System (fs) module to read HTML files
  • Basic routing to determine which page to show
  • Error handling to display appropriate messages when things go wrong
